Iris Dataset = 150 flower measurements
               labelled as 3 species

Your job = train a model that can identify
           the flower species from measurements alone

This is multiclass classification
on numeric features — not text

Perfect for beginners because:
→ Small (150 rows) — trains in milliseconds
→ Clean — no missing values, no noise
→ Balanced — 50 samples per class
→ Well-understood — you know what good looks like
→ Numeric features — no text featurisation needed
→ Real challenge — versicolor/virginica overlap
   makes it interesting but not impossible

Expected accuracy: 95-100% for good models
If you get below 90% → something is wrong
